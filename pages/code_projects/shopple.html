Description:
A web-app built using the MERN (MongoDB, Express, React, Node) stack that aims to provide a consolidated place for consumers to see ongoing in-store sales for a variety of retail stores.

Main Contributions:
Created the front-end UI design through prototyping with InVision Studio and implementing it with React. Utilized the gmail API in a node script that parses promotional emails for sale data and inserting it into the MongoDB database.

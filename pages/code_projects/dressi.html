Description:
A web-app built using React that was initially used for experimenting with API calls and displaying the data in the front-end. This app uses the openweatherapi to query the temperature of a city and currently uses a very simple condition to construct the google image search query that acquires the images to be displayed. The concept for this app is to help users find outfit inspiration depending on the weather of your city and future plans include implementing deep learning to make the search query more dynamic and specific.

Main Contributions:
Prototyped and created the front-end design using React and Javascript. Parsed user input for location and used it to make an API call to the openweatherapi to get the temperature data.
